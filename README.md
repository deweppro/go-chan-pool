# How to use it

```go
import pool "github.com/deweppro/go-chan-pool"

type Object struct {
	I int64
}

func main() {
	cpool := &pool.ChanPool{
		Size: 1, // pool size
		New: func() interface{} { // handler for creating a new object in pool
			return &Object{}
		},
	}
	a := cpool.Get().(*Object) //get object from pool
	cpool.Put(a) //set object to pool
}
```
